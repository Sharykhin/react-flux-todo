Install:
-------

```
npm install
```

Run:

```
npm start
```

Open in a browser:

```
npm run serve
```

Open browser on [http://localhost:8000](http://localhost:8000)

There is just light version and it going to be changed with Go (Jin framework) and mongodb on the server side
